# HR Analytics Dashboard
## 1.Project Title / Headline
📊 HR Analytics Dashboard: Employee Attrition & Workforce Insights
An interactive Power BI dashboard designed to analyze employee attrition, workforce demographics, department performance, and employee trends to support data-driven HR decision-making.

## 2.Short Description / Purpose
The HR Analytics Dashboard provides a comprehensive view of employee data, helping organizations understand attrition patterns, workforce distribution, salary trends, and employee demographics. The dashboard enables HR teams to identify key factors affecting employee retention and improve workforce planning strategies.

## 3.Tech Stack
The dashboard was built using the following tools and technologies:
- 📊 Power BI Desktop – Main platform for creating interactive dashboards and reports.
- 📂 Power Query – Data cleaning, transformation, and preparation.
- 🧮 DAX (Data Analysis Expressions) – Used for KPI calculations and custom measures.
- 🔗 Data Modeling – Established relationships among HR datasets for accurate analysis.
- 📄 Excel/CSV Dataset – Source data containing employee information.
- 📁 File Format – .pbix for development and .png for dashboard previews.

## 4.Data Source
Source: HR Employee Dataset
The dataset contains employee-related information such as:
- Employee ID
- Department
- Job Role
- Age
- Gender
- Salary
- Experience
- Attrition Status
- Job Satisfaction
The data is used to analyze workforce demographics, employee turnover, and departmental performance.

## 5.Features / Highlights
### Business Problem
Employee attrition can impact productivity, increase hiring costs, and affect organizational performance. HR departments need insights into workforce trends to improve employee retention and engagement.

### Goal of the Dashboard
To provide a centralized HR analytics solution that helps:
- Monitor employee attrition rates
- Analyze workforce demographics
- Identify high attrition departments
- Understand salary and experience-related attrition trends
- Support strategic HR decisions

### Key Insights Visualized
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate (%)
- Average Age
- Average Experience
- Attrition by Department
- Attrition by Salary Slab
- Attrition by Gender
- Attrition by Job Role & Satisfaction
- Age Group Distribution
- Experience-wise Attrition Trend
- Department-wise Employee Count

- Department Filter
- Age Group Filter

### Visualizations

- Job Role & Satisfaction Matrix
- Age Group Distribution Chart
- Gender-wise Attrition Pie Chart
- Experience-wise Attrition Trend Line
- Department-wise Employee Count Funnel Chart

## 7.Business Impact
This dashboard helps HR teams:
- Reduce employee turnover
- Improve workforce planning
- Identify departments with high attrition
- Enhance employee engagement strategies
- Make informed HR decisions using data-driven insights
